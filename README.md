# Portfolio

Ce projet a été réalisé avec [Angular CLI](https://github.com/angular/angular-cli) version 14.2.0, ainsi que DotNet
Pour lancer le projet en local:
```ng serve ``` à la racine du projet
puis se rendre à [localhost:4200](http://localhost:4200/).

## Projets
Ayant à développer deux projets à soumettre à un Jury lors d'une épreuve, un projet peut être trouvé [ici](http://localhost:4200/mes-projets/), le second projet étant ce portfolio.

### Diving Monitor
Pour l'instant, seul un projet de [Diving Monitor](http://localhost:4200/plongée/) est disponible. Celui-ci aura pour but d'aider
les plongeurs amateurs à tenir un carnet. Le carnet de plongée est obligatoire et chaque excursion doit y être consigné. Ce projet
a pour but de digitaliser ce carnet et d'aider les plongeurs à réaliser les calculs quotidiens (palier de sécurité, palier de décompression, calcul pour Nitrox), et également de remplir et sauvegarder ses données comme dans tout carnet de plongée classique agrégé par la FFESSM.
